Highlights

  NBT Editor

The most powerful NBT Editor in any open-source tool. Supporting S-NBT editing and extensions/plug-ins.

  Sculptor

Prune, copy, paste, and edit worlds without any external tools. The go-to feature for builders and building teams.

  World Converter

Convert worlds in seconds with the first open-source world converter that supports all platforms.

  GRF Editor

Edit Legacy Console GRF/GRH files with our easy-to-use GRF Editor, supporting S-GRF editing as well.

  Resource Converter

Have you ever wanted to convert a Java texture pack to Bedrock? Well, now you can! Supporting both JE & BE.

  More

Customization is key. You can fully customize the theme of the GUI to look like whatever you like.

Why Choose Minecraft Manipulator?


Compatible

Lodestone works everywhere, even on the web. The project is based on WASM, enabling it to reach everyone, including macOS, Windows, Linux, Android, and more. It will always be there when you need it.


Fun To Use

Lodestone offers extensive customization options while remaining user-friendly. This includes UI coloring customization through themes and customizable modules. Even with these features, the toolset remains easy to use


Fast & Powerful

Lodestone fosters community-driven expansion. With speed optimization and multi-threaded functionality, it ensures highly efficient performance, minimizing resource usage and eliminating memory leaks.


Smart Updates

The toolset will continually expand and evolve in sync with new Minecraft updates. All users will receive the latest updates simultaneously, seamlessly delivered over the air. No more worries about staying up-to-date.


Add-Ons

Extensions, plug-ins, and extension scripts are all supported within the Minecraft Manipulator! This enables users to extend functionality and build upon the modules within the toolset, making it fun for anyone to create for it!


Built-In Resigners

Built-in resigners enable you to access your legacy PS3/Vita worlds without the need for any third-party software. Region conversion is also included, supporting all legacy PS3/Vita regions. This makes it easier to convert across regions.


Accessibility

We want everyone to be able to use Lodestone. That's why we've implemented several settings and optimizations for users with visual disabilities. Ensuring accessibility is crucial to creating a world for everyone.


Translations

Supporting multiple languages enables us to connect with people worldwide, regardless of their location. We take pride in supporting all major languages. Being open-source, we also benefit from contributions from the community.


No Sketchy Servers

We do not hide secret servers or loggers of any kind inside of our work. We are 100% open-source and we highly encourage
the community to learn how the toolset works, pull requests are also encouraged!
